# Шрифты для Instagram Carousel Generator

Для корректной работы приложения необходимо скачать и разместить здесь файлы шрифтов Inter.

## Необходимые файлы:

1. `Inter-Regular.ttf` - обычный шрифт для основного текста
2. `Inter-Bold.ttf` - жирный шрифт для заголовков

## Где скачать:

1. **Google Fonts**: https://fonts.google.com/specimen/Inter
2. **GitHub**: https://github.com/rsms/inter/releases
3. **Официальный сайт**: https://rsms.me/inter/

## Инструкция по установке:

1. Перейдите на https://fonts.google.com/specimen/Inter
2. Нажмите "Download family"
3. Разархивируйте скачанный файл
4. Скопируйте файлы `Inter-Regular.ttf` и `Inter-Bold.ttf` в эту папку

## Структура папки после установки:
```
app/assets/fonts/
├── README.md (этот файл)
├── Inter-Regular.ttf
└── Inter-Bold.ttf
```

## Fallback:

Если шрифты не установлены, приложение будет использовать системные шрифты:
- Windows: Arial, Segoe UI  
- macOS: San Francisco, Helvetica
- Linux: Liberation Sans, DejaVu Sans

Но для лучшего качества рекомендуется установить оригинальные шрифты Inter.
